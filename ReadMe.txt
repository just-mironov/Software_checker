Релиз Version 0.1 
23.06.2020

Программа показывает версию windows, софт и версию софта.
Поиск софта производится в двух ветках реестра: 'Software\Wow6432Node\Microsoft\Windows\CurrentVersion\Uninstall', 
'SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\'
Отбор списка компьютеров происходит по маске "W[S,M,N]-....."
$SoftwareList в коде можно дополнять нужными программами.
Время выполнения на 1 компьютер 5-10 секунд.


Процесс запуска:
1) Запускаем скрипт (ПКМ -> Выполнить с помощью Powershell);
2) В первую колонку вносим список компьютеров;
3) Во второй колонке выбираем софт, который хотим проверить. Или снимаем все галочки, чтобы посмотреть весь софт;
4) Нажимаем проверить;
